Dasaran Perintah Git Dari CLI / Terminal

Create

  • git init: Dimana memulai sebuah project git untuk pertama kalinya
  • git clone: Intinya hanya untuk copy paste dari cloud ke cli kita

Sync

Note: Jika dalam git kita ingin memantau sebuah perubahan yang kita lakukan bisa gunakan git remote add origin "link github-nya", untuk mengcheck repo kita tersambung ke lokal bisa ketikan git remote -v

  • git push: Mengupload data dari lokal ke online
  • git pull: Sebaliknya menarik semua perubahan data dari online ke lokal
  • git Fatch: Mengambil resourch data dari server ke lokal tanpa adanya perubhan yg terjadi di server

Branch

pada saat melakukan branch pastikan kita sudah pindah pada branch tersebut dgn perintah "git checkout (nama branch-nya)"

  • git branch: Dimaana sebuah fitur coba-coba bisa di taro di sini dalam betnuk uji coba

    • dalam hal ini kita bisa melakukan perintah "git branch -D (nama branch-nya)" untuk menghapus branch tersebut akan tetapi kita harus pindah terlebih dahulu ke branch/main baru kita bisa menghapus branch tersebut
